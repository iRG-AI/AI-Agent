## 🎯 영상 기획서 설계: AI 에이전트 워크플로우 (Topic 1)

### 1. 최적 영상 포맷 설계 (Format Strategy)

| 요소 | 상세 전략 및 근거 | 데이터 기반 판단 |
| :--- | :--- | :--- |
| **영상 길이** | **8분 ~ 12분 사이.** 복잡한 워크플로우를 '완벽하게' 시연하려면 최소 8분은 필요하지만, 지루해지기 전에 이탈하는 것을 막으려면 12분을 넘기지 않아야 합니다. | 평균 시청 지속률(AVD) 데이터상, 10분 내외의 실습형 콘텐츠가 정보 밀도 대비 만족도가 가장 높음. |
| **구조 (Flow)** | **Hook (30초) → Problem/Pain Point (1분) → Core Concept (2분) → Step-by-Step Demo (5분) → Advanced Tip & Summary (2분) → CTA.** 이 구조는 20~70대 시청자가 정보를 습득하고 행동(따라 하기)으로 이어지도록 설계된 표준 포맷입니다. | 정보 전달형 콘텐츠의 AVD를 높이는 가장 검증된 패턴입니다. '단계별' 접근이 복잡성을 해소합니다. |
| **시각 자료** | 텍스트 설명 대신, **인포그래픽(Flowchart)**과 **화면 녹화(Screen Recording)**를 70% 이상 사용해야 합니다. 특히 AI 에이전트의 'Plan-Execute-Refine' 순환 구조는 시각적으로 명확하게 표현되어야 합니다. | 비개발자 타깃은 추상적인 설명보다, 눈으로 따라 할 수 있는 **구체적인 시연**에 반응률이 압도적으로 높습니다. |
| **톤앤매너** | **권위적이지만 친절함.** 기술 용어는 반드시 쉬운 비유(Analogy)를 사용하여 설명하고, '사장님/여러분' 호칭을 적절히 섞어 전문성과 친밀감을 동시에 확보해야 합니다. | 타깃 청중의 특성상, 너무 딱딱한 IT 전문가 톤은 이탈을 유발합니다. 실용적인 조언과 따뜻한 격려가 필요합니다. |

### 2. SEO 최적화 브리프 (Title & Description Brief)

#### A. 제목 (Title) 제안 (3가지 옵션)

**목표:** 검색 엔진 노출(SEO), 클릭 유도(CTR), 문제 해결 약속을 동시에 충족시켜야 합니다.

1.  **(최고 추천 - 실용성 강조)** **[AI 자동화] 코딩 없이 업무를 끝내는 AI 비서 구축법 (에이전트 워크플로우 완벽 가이드)**
    *   *분석:* '코딩 없이', '업무 끝내기', '완벽 가이드'라는 강력한 이득과 명확한 주제 키워드를 모두 포함합니다.
2.  **(호기심 자극형)** **당신의 시간을 3배 늘리는 AI 에이전트 비밀: 복잡한 업무를 스스로 처리하게 만드는 설계법**
    *   *분석:* '시간 절약'이라는 직접적인 이득을 강조하며, '비밀', '설계법'으로 콘텐츠의 깊이를 암시하여 클릭을 유도합니다.
3.  **(직관적 가이드형)** **AI 에이전트 10분 만에 배우기: 계획-실행-피드백 순환 구조 완벽 해부**
    *   *분석:* '10분 만에'라는 시간 제약과 핵심 개념('순환 구조')을 명시하여, 짧은 시간을 투자하는 시청자에게 매력적입니다.

#### B. 설명 (Description) 브리프

**목표:** 초반 3줄 내에 검색 키워드를 배치하고, 타임스탬프로 시청자가 원하는 정보를 즉시 찾게 하며, 최종 CTA를 명확히 제시합니다.
